# SQL Analytics Portfolio

Archived public portfolio of SQL analytics exercises.

This repository is retained as evidence of hands-on analytics engineering work:
query design, aggregation, ranking, retention analysis, marketplace analytics,
finance-style reporting, and operational data exploration.

## What Is Included

- Customer and city-level aggregation examples.
- Retention, session, ordering, and cohort-style query patterns.
- Product, merchant, salary, booking, and finance-oriented analytics prompts.
- Standalone SQL exercises preserved as historical work.

## Status

Archived portfolio work. The files are intentionally public to show the data
analytics foundation behind later security analytics, TrustOps, compliance, and
AI security engineering work.
