# SQL Analytics Portfolio

Archived public portfolio of SQL analytics and business problem-solving
exercises.

This repository is retained as evidence of hands-on analytics engineering work:
query design, aggregation, ranking, retention analysis, marketplace analytics,
finance-style reporting, and operational data exploration.

## Scope

- Customer and city-level aggregation examples.
- Retention, session, ordering, and cohort-style query patterns.
- Product, merchant, salary, booking, and finance-oriented analytics prompts.
- Ranking, top-N, average, distinct-count, and share-of-market logic.

## Structure

```text
sql/
  average-customers-per-city.sql
  user-retention-rate.sql
  product-market-share-at-the-product-brand-and-territory-levels.sql
  ...
```

Each file is a standalone SQL problem or analytics prompt. The repository name
uses "portfolio" because the value is the query reasoning pattern: turning a
business question into readable SQL.

## Status

Archived portfolio work. The files are intentionally public to show the SQL and
analytics engineering foundation behind later security analytics, TrustOps,
compliance, and AI security engineering work.
